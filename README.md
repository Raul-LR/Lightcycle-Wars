
# Lightcycle Wars

## Overview

Lightcycle Wars is a personal indie project developed in Unity, inspired by the TRON universe and heavily influenced by the classic Linux game GLtron.

The goal of this project is to modernize the classic lightcycle gameplay experience using current game development technologies, modern rendering techniques, and contemporary gameplay design.

Players control futuristic light bikes capable of generating solid energy trails while moving through the arena. The objective is to eliminate opponents by forcing them to collide with these trails while avoiding collisions yourself.

---

# Features

## Current Gameplay Features

* Keyboard and gamepad support
* Free camera movement
* Modernized vehicle controls inspired by contemporary games such as GTA
* Multiplayer-oriented gameplay
* Multiple lives system
* Dynamic light trail generation
* Modern visual effects and rendering techniques
* Fast-paced arcade gameplay

---

# Project Status

## Early Development / Proof of Concept

Lightcycle Wars is currently under active development and is still considered an early prototype / proof of concept version.

At this stage, the project mainly contains the core gameplay mechanics required to validate the main gameplay loop and technical foundations of the game.

### Currently Implemented

* Basic player movement
* Simple start and game over menus
* Trail collision system
* Basic visual effects
* Basic sound effects
* Camera and input systems

### Planned Features

A significant amount of work is still pending, including:

* Gameplay balancing
* Advanced multiplayer features
* User interface improvements
* Menus and settings systems more complex
* Better audio and fx implementation
* Improve 3d models using the .FBX instead of .obj ones and more optimized ones
* Add a good textures for the 3d lightcycle model, with different colors for each player
* Add a lightcycle color selection
* Matchmaking systems
* Additional game modes
* Better visual polish
* Performance optimization
* Network optimization
* AI opponents
* Improved animations and effects
* Dedicated server support
* Cross-platform improvements

The project is continuously evolving and many systems are still experimental or subject to redesign.
